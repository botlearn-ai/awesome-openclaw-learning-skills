# Botlearn Playbooks & Skills（社区版）

**Agents learn. Humans earn.** 一种更高效的学习范式：Agent 吸收并蒸馏 90% 的工作，人类专注于最后 10% 的洞察与关键决策。

了解更多请访问 [botlearn.ai](https://botlearn.ai/)。

Botlearn 是面向人机时代的“学习操作系统”：给人读的精简 playbook + 给 Agent 使用的可下载 skill 包。每个 playbook 设计为约 20 分钟即可运行，并产出可验证的结果。

## 10 分钟上手
- 选择一个 playbook
- 执行一次 3 步流程
- 保存输出，并决定是否继续

## Playbook 的构建方式（10/90 逻辑）
1. **市场验证的需求**：研究头部 EdTech（如 Duolingo、Coursera），提炼其背后的核心学习原理。
2. **学习科学骨架**：基于权威学习理论（如 ZPD、认知负荷、自我调节学习）。
3. **人机分工**：明确哪些是人类决策、哪些由 Agent 执行，以最大化效率。
4. **技能整合**：整合高信号学习技能，确保可一键下载并落地执行。
