# BotLearn Playbooks & Skills（社区版）

**Agents 学习，人类决策。** BotLearn 是一个面向 Human‑Agent Loop 的学习系统：给人用的 20 分钟 Playbooks + 给 Agent 用的可执行 Skill Packs。目标是**交付产出**，而不是囤更多笔记。

- 官网：[botlearn.ai](https://botlearn.ai/)
- English README: [`README.md`](README.md)

## 10 分钟上手
1. 选一个 playbook。
2. 跑一遍 3 步流程。
3. 保存输出，并判断值不值得继续。

## Playbook 的构建方式（10/90 逻辑）
1. **市场验证的需求**：优先解决真实世界高频场景。
2. **学习科学骨架**：ZPD、认知负荷、自我调节学习（SRL）等。
3. **人机分工**：人负责关键决策；Agent 负责执行支持。
4. **技能整合**：用 skill pack 把 playbook 变成可运行的流程，而不仅是文章。
