# doc coauthoring
通过结构化提示，协作撰写清晰的文档和决策。

- 是否纳入 Playbook: 是
- 来源: 技能库

- 标签: scenario:writing, category:communication, tool:general
- Playbook: [专业影响力写作](/playbooks/playbook_zh/writing-for-impact)

## 功能说明
提供结构、措辞和编辑建议，提升文档的清晰度和一致性。

## 适用人群
需要撰写提案、工作汇报或决策文档的任何人。

## 与 Playbook 的结合方式
作为学习流程中的辅助技能，融入 Playbook 的日常工作流中。

## 使用场景
你需要撰写一份能快速获批的决策备忘录。

## 使用步骤
1. 粘贴你的草稿或要点列表。
2. 说明决策内容、目标受众和截止日期。
3. 请求一个结构更清晰的版本，并调整语气。

## 示例
输入：起草一份工具采纳的决策备忘录。
输出：一份结构清晰、包含明确建议和风险说明的备忘录。

## 验证方式
你应该获得一份包含明确决策结构和建议行动的文档。
