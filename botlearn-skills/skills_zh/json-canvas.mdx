## 功能说明
Helps generate a visual node map for concepts and connections.

## 适用人群
Learners who need visual maps of ideas.

## 与 Playbook 的结合方式
Fits into the playbook routine as a supporting skill for the learning flow.

## 使用场景
You want to visualize a topic map for a new subject.

## 使用步骤
1. List the concepts you want to connect.
2. Ask for a canvas node structure.
3. Paste the result into Obsidian Canvas.

## 示例
Input: Map key ideas about active recall.
Output: Canvas nodes and links suggestion.

## 验证方式
